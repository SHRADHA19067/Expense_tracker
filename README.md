# Smart Expense Tracker (Python)

A simple Python program to record, store, and analyze personal expenses using JSON file handling.

## Features
- Add, view, filter, and sort expenses
- Auto-save expenses to JSON file
- Summary report (total, highest, lowest)
- Export data to CSV
- Visualization using Matplotlib (bar, pie, line charts)

## Technologies Used
- Python
- JSON
- Pandas
- Matplotlib

## How to Run
1. Clone the repository  
2. Run: python smart_expense_tracker.py  
3. The program will auto-load expenses.json (if available)
4. Follow the menu options to add/view/analyze expenses
